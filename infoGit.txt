0 - CONFIGURAR LOCALMENTE GIT :
   * git config --global user.name "nombre de usuario"
   * git config --global user.email "email"

1 - Crear repositorio local ->  git init

2 - Para ignorar archivos no deseados se crea el archivo ".gitignore"
   * Dentro colocar el nombre de lo que se quiere ignorar. (nombre de archivo o nombre de directorio)


3 - Agregar a seguimiento "git add ."
4 - Comprobamos estado del repositorio "git status"
    * Si todo en verde queda solamente el "commit"

5 - Realizar el commit 'git commit -m "comentario"'
6 - Comprobamos estado del repositorio "git status"
    * Si te muestra este mensaje "On branch master nothing to commit, working tree clean" ya esta todo comiteado

7 - Enlazar con el repositorio remoto "git remote add origin (urlRepositorioRemoto)"
8 - Comprobar la vinculaci√≥n local con la remota "git remote -v"


// EXTRA - Cambiar el nombre de la rama actual //
git branch -m "nuevo nombre"

9 - Subimos al repositorio remoto todo los cambios "git push origin clase-apoyo-1"
10 - Realizar un pull para traer los nuevos cambios
